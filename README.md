# Reyyz
BOT WHATSAPP VIA TERMUX BY Reyyz

### Alat dan Bahan
Siapin alat dan bahannya.
```bash
> niat
> 2 handphone (1 buat jalanin sc, 1 buat scan qr)
> internet yang memadai
> aplikasi whatsapp
> aplikasi termux
> cemilan
```

### Cara Installnya
Sebelum lu jalanin sc nya install dulu lah.
```bash
> kalo lu belum punya apk termux, download di playstore
> masuk ke apk termux lalu ketik dibawah ini!
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> git clone https://github.com/palabapakmu/Reyyzbot
> cd Reyyzbot
> bash install.sh
> node index.js
> Tinggal scan qr dah
```
